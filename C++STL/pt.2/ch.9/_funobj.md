# STL 함수 객체   
* * * 
## 1. 함수 객체의 종류   
operator() 연산자를 오버로딩한 클래스 객체   
함수자라는 애칭으로 더 많이 사용됨   

일반 함수 객체: 특정 기능을 수행하는 함수 객체   
함수 어댑터 : 함수류를 인자로 받아 다른 함수 객체로 변환   

함수자를 쓰는 이유?   
특정 상황에서 일반 함수보다 빠르다(다른 함수의 인자로 전달될 때)   
상태를 가지는 함수(멤버 데이터가 있고, 멤버 함수를 활용 가능)   
'함수처럼 동작하는 객체'이므로 다른 멤버 변수와 멤버 함수를 가질 수 있고 일반 함수에서 하지 못하는 지원을 받을 수 있다.   
속도도 일반 함수보다 함수 객체가 빠르다   
* * * 
## 2. 산술 연산 함수 객체
STL은 여섯 가지 산술 연산 함수 객체 제공   

- plus<T> : 이항 연산 함수자로 + 연산   
- minus<T> : 이항 연산 함수자로 - 연산   
- muliplies<T> : 이항 연산 함수자로 * 연산   
- divides<T> : 이항 연산 함수자로 / 연산   
- modulus<T> : 이항 연산 함수자로 % 연산   
- negate<T> : 단항 연산 함수자로 - 연산   

* * *
## 3. 비교 연산 조건자
STL은 여섯가지 비교 연산 함수 객체 조건자를 제공   
- equal_to<T> : 이항 조건자로 == 연산
- not_equal_to<T> : 이항 조건자로 != 연산
- less<T> : 이항 조건자로 < 연산
- less_equal<T> : 이항 조건자로 <= 연산
- greater<T> : 이항 조건자로 > 연산
- greater_equal<T> : 이항 조건자로 >= 연산

* * *
## 4. 논리 연산자
- logical_and<T> : 이항 조건자로 && 연산
- logical_or<T> : 이항 조건자로 || 연산
- logical_not<T> : 단항 조건자로 ! 연산
