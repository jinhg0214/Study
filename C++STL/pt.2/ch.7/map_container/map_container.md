<h1> Map 컨테이너 </h1>

- 연관 컨테이너중 가장 자주 사용하는 컨테이너
- 원소를 key와 value의 쌍으로 저장
- key는 유일한 원소
- map은 [] 연산자를 제공하여 원소의 value에 쉽게 접근 가능함
- 원소는 pair 객체로 저장되며, first는 key로 second는 value로 저장됨
- insert() 멤버함수는 pair 객체를 인자로 받아 map 컨테이너에 삽입함

- 연관 컨테이너, 노드 기반 컨테이너
- 원소(key)가 자동 정렬되며, set과 다르게 (key, value)의 쌍으로 이루어짐
- 빠른 원소 검색(로그 시간 검색 복잡도)
