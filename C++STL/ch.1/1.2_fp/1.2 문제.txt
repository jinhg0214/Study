1. 다음은 Print()의 주소를 저장하는 함수 포이넡의 정의입니다. 빈칸을 채우세요

    void Print(int data){ ... }

1) Print() 함수가 정적 함수나 전역 함수라면 함수 포인터는 (    ) 와 같이 선언합니다.
2) Print() 함수가 MyClass 클래스 내의 멤버 함수라면 함수 포인터는 (    ) 와 같이 선언합니다.
3) 함수 포인터 pf가 전역 함수 Print()의 주소라면 (    ) 처럼 전역 함수를 호출합니다.
4) 함수 포인터 pf가 MyClass의 멤버 함수 Print()의 주소고, 객체가 obj라면 (    )처럼 멤버 함수를 호출합니다.

/*
void (*pf)(int)
void (MyClass::*pf)(int)
pf(10) or (*pf)(10)
(obj.*pf)(10)
*/
