
<h1> List 컨테이너 </h1>

- List는 시퀸스 컨테이너, 노드 기반 컨테이너

-순차열 중간에 삽입, 제거가 빈번하게 발생하고,
원소의 상대적 순서가 중요한 경우 list 컨테이너 사용.

- 이중 연결 리스트 구조, 원소를 앞, 뒤에 모두 추가할 수 있음.

- 시퀸스 컨테이너이므로, 가장 첫 원소와 마지막 원소의 참조인 front(), back() 사용 가능

- 배열 기반이 아니므로 []연산자나 at() 멤버함수는 제공하지 않음.

- 노드 기반이므로 insert()나 erase()는 배열 기반보다 효율적으로 동작함(메모리 재할당 없이 동작)

- 노드 기반 컨테이너 이므로 양방향 반복자 지원. 

- 값의 비교로 원소를 제거하는 remove(), remove_if()를 제공함

- list는 다른 list와 결합할 떄 좋은 컨테이너

- splice() 멤버함수는 노드의 연결 연산만으로 두 list를 하나의 리스트로 결합 할 수 있음

- merge() 멤버함수는 두 list를 하나의 list로 정렬하며 결합함.

- 이외에 reverse(), sort(), unique()등의 멤버함수 제공
